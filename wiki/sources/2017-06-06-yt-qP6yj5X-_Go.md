# How To Make Your Type Look Distressed and Add Texture PSD Tutorial

## Summary

A short Photoshop technique tutorial. **Chris Do teaches it himself** — he opens with
"Hey guys, Chris here for the process," and narrates the whole walkthrough in the first
person, so attribution is unambiguous (no `uncertain` flag needed).

The tutorial demonstrates how he added a distressed/photocopier texture to the wordmark
for the "Spartan logo design challenge." The workflow: copy the type set, paste it into a
pre-built textured PSD template purchased from **Creative Market** (a smart-object /
smart-layer file, referenced as being from "Rob Brink"); double-click the smart object,
paste the type in, scale it to match, then add two horizontal bars (top and bottom) drawn
with the marquee tool to "anchor" the design; adjust spacing/leading; save the smart
object so the built-in texture and outline effect re-render onto the type; export as PNG
and bring it back into the working file. He notes the whole logo took "about 30 minutes."

This is **near-zero persona value** — it is procedural Photoshop craft (keyboard shortcuts,
smart objects, marquee fills), not business, mindset, or teaching philosophy. Recorded as a
context page only. Domain tagged `design-craft` for corpus completeness.
