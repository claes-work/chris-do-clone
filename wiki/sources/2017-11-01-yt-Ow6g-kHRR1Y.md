# Master the Pen Tool in Adobe Illustrator

## Summary

A hands-on software tutorial in which Chris Do demonstrates tracing a letterform
(the letter "G") with the Adobe Illustrator pen tool. Content is ~90% software
mechanics: converting a professionally-drawn letter to outlines, reading its
anchor points, and re-drawing the curve with the pen tool using shift (constrain)
and option (convert anchor / drop handle) modifiers. Chris folds in a few
design-craft principles about how to draw good curves and why studying
masterfully-drawn letterforms improves your eye. Low persona value overall; useful
mainly as design-craft procedure and for a couple of Chris's working-method asides.

## Key claims (by speaker)

### Chris Do — pen tool / Illustrator mechanics
- `[2017-11-01]` Command+Shift+O outlines type so you can study how a
  professionally drawn letter is constructed. (paraphrase)
- `[2017-11-01]` Professional letterforms use as few anchor points as possible,
  placing each point at the extremity/tangent farthest away (top and bottom of an
  arc). (paraphrase)
- `[2017-11-01]` Hold Shift while dragging to constrain a handle to the horizontal/
  vertical axis; hold Option and click a point to drop its handle and create a
  sharp corner / start a new curve direction. (paraphrase)
- `[2017-11-01]` Command+K opens Preferences to change the guide display to dots/
  dashes/grids. (paraphrase)
- `[2017-11-01]` Command+Option+J with points selected, then "horizontal," snaps
  the selected points into alignment (one straight stroke across). (paraphrase)
- `[2017-11-01]` A postscript / Illustrator file describes a visual thing using
  math; you can open an Illustrator file in a text editor (Word) and read/modify
  the underlying code — people doing generative "processing" work do this. (paraphrase)

### Chris Do — design-craft principles
- `[2017-11-01]` Drag the pen-tool handle in the direction the curve is moving —
  that resolves the common beginner confusion of whether to drag up or down. (paraphrase)
- `[2017-11-01]` The Bezier interface is just a human-friendly way of telling the
  math where the curve should head before it reaches the next point, instead of
  typing code. (paraphrase)
- `[2017-11-01]` Watch for "weird flat spots" in a curve and adjust the point/
  handle so the transition happens more smoothly; judge quality by looking at the
  inside shape of the letter and how smooth it reads. (paraphrase)
- `[2017-11-01]` Studying masterfully-drawn letterforms trains your eye for good
  curves. (paraphrase)

### Chris Do — working-method asides (persona-relevant)
