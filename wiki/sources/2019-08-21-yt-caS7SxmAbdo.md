## Summary

An unnamed Futur team member shares three project-management tips for getting team work
delivered on time: (1) kick off every project with a clear one-page brief, (2) set
expectations and get buy-in from the person doing the work rather than barking orders, and
(3) set internal deadlines with 1–2 days of padding before the client-facing deadline. The
video is a promo for The Futur's "Practical Project Management" course and is sponsored by
Monday.com.

## Key claims

_(All attributed to the unnamed Futur instructor, NOT Chris Do — context only.)_

- [2019-08-21] Tip 1 — Start every project with a clear one-page brief instead of just
  forwarding the client's email; the manager must interpret the client info and translate
  it for the team. The brief should include the project goal, the context/where it will be
  shown, who it's for, the creative parameters to work within, and clear task assignments
  per team member. It lives wherever the project files live (e.g. the Dropbox project
  folder). (paraphrase)
- [2019-08-21] Tip 2 — Set expectations and get buy-in; don't bark orders. A team eager to
  please will say "yes" to a Thursday deadline even when they don't think it's possible, so
  leave room for them to push back. (paraphrase)
- [2019-08-21] Suggested assignment script: "By Thursday I'm hoping to have three versions
  of this design to review — does that sound manageable? Do you have any questions?" —
  giving the team member space to say no yields transparency and a better gauge of the real
  effort. (paraphrase; script demonstrated)
- [2019-08-21] When you don't know how long a task takes (e.g. managing a discipline you
  don't practice), ask the expert doing it for their own estimate rather than dictating one.
  Buy-in makes the person hold themselves accountable. (paraphrase)
- [2019-08-21] Tip 3 — Set internal deadlines with padding: schedule the team's internal
  delivery 1–2 days before the client/stakeholder deadline (e.g. internal Thursday morning
  for a Friday client deadline) so the manager has a 24–48h buffer to review, give feedback,
  and course-correct instead of scrambling the night before. (paraphrase)
- [2019-08-21] Avoid scheduling client deliveries/posts on Fridays; switch them to Mondays
  so, at worst, you have the weekend as a fallback. (paraphrase)
- [2019-08-21] Plug: The Futur's "Practical Project Management" course. Sponsor: Monday.com
  (code "future" / URL in description for 10% off). (paraphrase; promotional)
